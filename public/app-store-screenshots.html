2. App Review Notes (CRITICAL - Copy this to App Store Connect)

Dear App Review Team,

DRVN is a garage management app designed for small and medium auto repair shops in Israel. While it uses a WKWebView for its UI layer, it provides significant native functionality that goes well beyond a simple website wrapper:

NATIVE FEATURES:
1. Native iOS Push Notifications (FCM via APNs) - Real-time appointment reminders sent every 15 minutes via Firebase Cloud Functions → FCM → APNs. Users receive native iOS notifications even when the app is closed.

2. Offline Support - Full Service Worker implementation with network-first caching for HTML and cache-first for images. The app works completely offline with an offline fallback page.

3. Camera & Photo Integration - Users can capture photos of vehicles directly from the app using the native camera, with images stored in Firebase.

4. Location Services - Integrated location access for garage-related functionality.

5. Background Processing - Cloud Functions run server-side to process reminders and send notifications on schedule.

6. App Shortcuts - Quick actions from the home screen (Add Vehicle, View Calendar).

7. Deep Link Support - URL scheme support for direct navigation to specific app sections.

WHY NATIVE APP vs WEBSITE:
- Push notifications require native APNs integration (Safari web push is limited on iOS)
- Offline functionality provides reliable access in garage environments with poor connectivity
- Camera access is more seamless in the native context
- Home screen presence and app shortcuts improve daily workflow for mechanics
